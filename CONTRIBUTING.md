# Contributing to Content Automation Kit

Thank you for your interest in contributing! This is a community-driven project.

## How to Contribute

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes**
4. **Test your changes**
5. **Commit with clear messages**: `git commit -m "Add: description of change"`
6. **Push to your fork**: `git push origin feature/your-feature-name`
7. **Open a Pull Request**

## Areas We Need Help

- Platform integrations (TikTok, YouTube, Pinterest)
- Video templates (Manim, Motion Canvas, Remotion)
- Documentation and tutorials
- Test coverage
- Analytics dashboard
- Multilingual support

## Code Style

- Python: Follow PEP 8
- JavaScript: Use Prettier
- Document your functions
- Add type hints where possible

## Questions?

Open an issue or start a discussion. We're friendly!
